# Portfolio Website

#### This project used with django and bootstrap

## Start with
#### 1. Setting environment 
```
python -m venv myvenv
source myvenv/bin/activate 
```
#### 2. Install django inside the project
```
pip install django
```
#### 3. Run th program
```
python manage.py runserver
```
